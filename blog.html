1) What is DOM and what is the purpose of DOM?

Ans: The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.
DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove elements from a website document. DOM manipulation is when you use JavaScript to add, remove, and modify elements of a website

2)How will you select HTML elements using DOM? Name the DOM methods.?

Ans: There are 5 ways in which you can select elements in a DOM using selectors.

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

    • 3) What is event bubble?

    Ans: Event bubbling is a method of event propagation in the HTML DOM API when an event is in an element inside another element, and both elements have registered a handle to that event.

    4) What is a callback function and why will you use it?

    Ans: A callback function is one of the superpowers of JavaScript. It is the way JavaScript passes a function into another function as an argument. The callback function is called in the outer function to execute an action.
